# 项目简介

本项目为全栈开发实践项目，包含以下三个主要子模块：

- **api-springboot**：基于 Spring Boot 的后端服务，提供 RESTful API。
- **web-vue**：基于 Vue 3 + Vite 的前端单页应用。
- **wx-miniprogram**：微信小程序端，包含页面、样式、配置等。
