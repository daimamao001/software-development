## 1 前端

### 1.1 安装依赖与启动
```bash
cd web-vue
npm install
npm run dev
```

### 1.2 打包

```bash
npm run build
```

生成 `dist/`，即静态可部署文件。


## 2 后端

### 2.1 使用 Maven 运行

在项目根目录执行：

```bash
mvn spring-boot:run
```

### 2.2 打包成 Jar 文件

```bash
mvn clean package
```

生成的文件在 `target/` 目录下，例如：

```
target/demo-0.0.1-SNAPSHOT.jar
```

### 2.3 上传至服务器

使用 `scp` 将文件上传至服务器：

```bash
scp target/demo-0.0.1-SNAPSHOT.jar user@your.server.ip:/path/to/
```

### 2.4 服务器启动项目

在服务器上执行：

```bash
java -jar demo-0.0.1-SNAPSHOT.jar
```

后台运行方式：

```bash
nohup java -jar demo-0.0.1-SNAPSHOT.jar > app.log 2>&1 &
```

查看日志：

```bash
tail -f app.log
```
